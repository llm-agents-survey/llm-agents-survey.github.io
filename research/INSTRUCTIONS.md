# Research Instructions

Each sub-agent writes a structured markdown file to this directory.

## Output Format

Each file should be named `<topic>.md` and contain:

```markdown
# <Topic> - LLM Agents Survey Research

## Overview
Brief synthesis of the landscape in this area.

## Key Papers / Preprints
For each item:
- **Title** (Year) - Authors
  - URL: <url>
  - Summary: <2-3 sentence summary of key contribution>
  - Key ideas: <bullet points>

## Blog Posts & Articles
- **Title** (Date) - Author/Org
  - URL: <url>
  - Summary: <1-2 sentences>

## GitHub Repos & Tools
- **Name** - <description>
  - URL: <url>
  - Stars/activity: <info if available>

## Taxonomy / Key Concepts
Synthesize the key ideas, distinctions, and concepts in this area.

## Timeline of Major Milestones
Chronological list of major developments.
```
